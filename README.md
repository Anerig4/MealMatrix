# 🍽️ MealMatrix AI

AI-powered meal recommendation system based on user mood.

## Features
- Mood-based food suggestions
- Node.js + FastAPI integration
- Simple frontend UI


### AI Backend
cd fastapi-backend
pip install -r requirements.txt
uvicorn main:app --reload

### Node Backend
cd api
npm install
node server.js

### Frontend
Open public/index.html
